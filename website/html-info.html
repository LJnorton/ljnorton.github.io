All About HTML

What is HTML?

HTML is the foundation of all webpages. It stands for Hypertext Markup Language, and is the language used to establish the basic structure of a web page, including things like headings, titles, paragraphs, images, and what elements are going to be in a page, such as headers and navigation bars.


The History of HTML

HTML was first invented by a man called Tim Berners-Lee, when he worked as a physicist at CERN. This first version was published back in 1991, and had only 18 tags. It has since grown and been developed over the last three decades, to HTML5 - and has become a web standard.


How does HTML work?

HTML pages are made of up tags, or elements, which can be used to establish a hierarchy on a page, which is useful for structuring all of the features on a web page into a nice, ordered list.
