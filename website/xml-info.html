All About XML

What is XML?

Like HTML, XML is a markup language. However, XML is intended primarily for storing and sharing data, unlike HTML, which is designed to display data to the end user. It stands for eXtensible Markup Language.


How does it differ to HTML?

XML differs from HTML in several ways. First of all, where HTML has a list of predefined tags, XML does not. This means that it is down to the author of any XML file to define any and all tags they wish to use within the XML file. Furthermore, XML is Case Sensitive. There can be any number of tags within an XML file, so long as they are properly defined.
