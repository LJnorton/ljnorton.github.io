All About Javascript

What is Javascript?

Javascript is a programming language within web design, which allows for use of more complex or advanced features to a web page, such as a live clock or animated graphics on a page.


What can Javascript do?

As earlier mentioned, Javascript can be used for a wide range of dynamic functions on a website. These can often be used to make the page pop a lot more, adding an interactive and dynamic layer to an otherwise static page.
